# tf-imle
Tensorflow implementation and Jupyter notebooks for Implicit Maximum Likelihood Estimation


## Requirements: 
* tensorflow==2.3.0 or tensorflow-gpu==2.3.0
* numpy==1.18.5
* matplotlib==3.1.1
* scikit-learn==0.24.1
* tensorflow-probability==0.7.0
